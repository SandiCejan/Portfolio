Visual Studio

Visual Studio is an IDE, made for developing games, websites and applications for windows, iOS, Android, Linux, tvOS, etc. You can use C#, C++, javascript, Visual Basic, and many more programming languages. It is very extendable, including a lot of extensions and packages. It was made by Microsoft corporation, first released in 1997, and updating ever since. The latest version is Visual Studio 2022, It can be installed from Visual Studio Installer, downloaded from Microsoft official website. It has a very nice UI, it is very powerful, and it can be used as a debuger for websites, unity game development, windows forms applications, etc.

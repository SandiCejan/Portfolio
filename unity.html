Unity

Unity is a game development software, that works on C# programming language. It is very beginner friendly, easy to understand and very extendable. You also have a free shop, called Unity asset store, where you can get some resources and packages you need to make a game. It has a very nice UI. It is very optimized in 2D rendering, but not optimized for 3D and VR.

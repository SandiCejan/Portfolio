Visual Studio Code

Visual Studio Code is IDE, made by Microsoft corporation, first released in 2015. It is very popular for making websites and applications. You can use all kinds of programming languages. It is extendable, including a lot of extensions and packages. The latest version is Visual Studio Code 2022. It can be installed and downloaded from Microsoft's official website. It has a very nice UI, and it can be used as a debuger for websites and other kinds of projects.
